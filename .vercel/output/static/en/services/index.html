Development & Technologies

Frontend Development

The website interface is implemented using JavaScript and React, ensuring a flexible and dynamic user experience. If the website requires static optimization or server-side rendering, we use Next.js to improve speed and SEO. HTML and CSS provide a solid foundation and styling for the website, while Astro enables fast loading of lighter static pages.

Content Management

Sanity serves as a flexible CMS that allows you to manage content independently. Sanity integrates seamlessly with our development environment and React-based frontend code.

Hosting and Deployment

Vercel serves as the primary hosting platform, providing powerful tools for publishing and version control. It also allows us to test and implement new updates smoothly and quickly without downtime.

Database and Data Storage

Depending on the project's requirements, we can use various database solutions, such as NoSQL and SQL databases, ensuring scalability and data security.

3D Development

We create 3D content using Houdini and Three.js, which enable visually impressive and interactive three-dimensional elements on your website. Houdini provides powerful tools for creating 3D models and animations, while Three.js allows seamless integration with the website, enhancing the user experience and adding visual depth.

We combine these tools and technologies to ensure your website meets all the requirements of the modern web.

GDPR Compliance

Matomo only collects the necessary data, allowing you to adjust data collection precisely and ensuring user privacy is respected. In addition, Matomo ensures that data stays within the EU, strengthening your site's GDPR compliance.

Analytics Management and Reporting

Matomo offers comprehensive reporting tools, allowing you to monitor traffic and behavior closely for site improvements. All analytics are accessible on your own dashboard, where you control and retain collected data without third-party involvement.

Integration and Privacy Notices

Adding Matomo is straightforward, and we assist you in creating clear privacy notices and cookie settings for the website, allowing users to manage their data usage. This GDPR-compliant analytics solution ensures that you can responsibly leverage valuable data without concerns about breaching data protection regulations.

Accessibility

We ensure website accessibility with the following measures.

  • • Alt texts for images
  • • Adequate color contrast between text and background
  • • Full keyboard navigation
  • • ARIA labels to clarify UI elements
  • • Logical heading structure and semantic HTML tags
